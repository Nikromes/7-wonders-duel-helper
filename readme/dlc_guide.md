# Руководство по добавлению DLC

## Как добавить новое DLC

### 1. Данные в `data.js`

Добавьте поле `dlc: "НазваниеDLC"` к жетонам и чудесам:

```javascript
// Жетоны
{ title: "Название жетона", desc: "Описание", tags: [...], color: "token", cost: [], dlc: "НазваниеDLC" }

// Чудеса
{ title: "Название чуда", desc: "Описание", tags: [...], color: "wonder", cost: [...], dlc: "НазваниеDLC" }
```

### 2. Стиль бейджа в `styles.css`

Добавьте CSS-класс `.dlc-названиеdlc` (lowercase):

```css
.dlc-названиеdlc {
    background: linear-gradient(135deg, rgba(R, G, B, 0.7), rgba(R, G, B, 0.7));
    color: #fff;
    border: 1px solid rgba(R, G, B, 0.4);
}
```

### Существующие DLC

| DLC | CSS-класс | Цвет |
|-----|-----------|------|
| Pantheon | `.dlc-pantheon` | Фиолетово-золотой |
| Agora | `.dlc-agora` | Бирюзовый |

## Сортировка

Элементы автоматически сортируются: **базовая игра** → **DLC (по алфавиту названия DLC)** → внутри группы по алфавиту названия.

## Карты эпох из DLC

Если DLC добавляет новые карты в колоды эпох, добавляйте их в соответствующий массив `predictorDeck[1|2|3]` в `data.js` с полем `dlc`.
